Detection App

Instructions and Overview

Alpha numeric app webapplication provides you with a facility to detect number or alphabets by using self trained models.It makes use of two models first is for box detection and second is for digit or alphabet recognition.

For instance To extract Meter Reading and Meter Number :

  • Meter Number is usually denoted in a smaller size as compared to other visual text elements present in the meter images and thus, we typically filter the region representing the whole meter number as whole followed by recognition of digits in filtered region.

  • When meter reading digits are large enough in resolution such that they are more visually distinguished from each other as well as a proper bounding box can be drawn over the digits with minimal overlapping of visual region of other digits.

How to use the Alpha Numeric app :

  • Create a new project.

  • Add the meter box images in the project.

  • Add images to box detection model and make boxes on the area of readings and train the model.

  • Now train the digit recognition model.

  • Once the box detection model is trained go to digit Recognization model and tag the numbers or alphabets to be detected.

  • After the training is completed, click on "Use Model". Now, you are ready to auto detect the meter readings.
